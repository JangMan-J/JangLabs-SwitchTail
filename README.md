# SwitchTail

**The operator's switchboard for agentic terminals — a Zellij plugin.**

SwitchTail is a window-manager-grade control plugin for [Zellij](https://zellij.dev),
built for running fleets of agentic development terminals (Claude Code and
friends): one-press focus to any line by ID, hot-seat swap into the main seat,
per-line message send/receive, and a live call-log of everything the fleet does
— monitored, triaged, categorized, sorted, without leaving the board.

Vocabulary (retro telephony): the Zellij session is the **exchange**; a tab of
agent panes is a **board**; one agent pane is a **line**; the focused working
position is the **seat**; N parallel lines are a **trunk**; assembling lines
into a board is **patch**ing; the operator's live event feed is the
**call log**; a line that wants attention is **ringing**.
